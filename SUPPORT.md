# Támogatás / Support

## Technikai segítség / Technical support

- Reprodukálható hibához használd a bug report sablont.
- Használati kérdéshez vagy ötlethez használj GitHub Discussiont, ha engedélyezve van.
- Biztonsági hibát kizárólag a [SECURITY.md](SECURITY.md) szerint jelents.
- Use the bug template for reproducible defects.
- Use GitHub Discussions for usage questions when enabled.
- Report vulnerabilities only through [SECURITY.md](SECURITY.md).

A közösségi támogatás best-effort, nem garantált válaszidejű és nem helyettesít
adózási, jogi vagy könyvelési tanácsadást. A VIES-válasz üzleti értelmezéséért és a
helyi jogszabályok betartásáért az integráló fél felel.

Community support is best-effort, has no guaranteed response time, and is not tax,
legal, or accounting advice. Integrators remain responsible for business decisions
and local compliance.

## Anyagi támogatás / Financial support

Az adomány önkéntes, nem vásárol prioritást, SLA-t, biztonsági kivételt vagy döntési
jogot. Ha a projekt hasznos, itt hívhatod meg a fejlesztőt egy kávéra:
[buymeacoffee.com/lgx0](https://buymeacoffee.com/lgx0). A GitHub Sponsor gombot a
`.github/FUNDING.yml` kezeli.

Donations are voluntary and do not buy priority, an SLA, security exceptions, or
governance rights. If the project helps you, you can
[buy the developer a coffee](https://buymeacoffee.com/lgx0). GitHub's
`.github/FUNDING.yml` controls the Sponsor button.
