fájlok jelölését, és kifejezett szabadalmi engedélyt tartalmaz.

The project uses the **Apache License 2.0** (`SPDX: Apache-2.0`). It permits private,
commercial, and open-source use, modification, and distribution while requiring
license/attribution preservation, notices for changed files, and providing an
explicit patent grant.

Miért nem MIT? Az MIT rövidebb és szintén megengedő, de nem tartalmaz az Apache-2.0-
hoz hasonló részletes, kifejezett szabadalmi rendelkezést. Egy integrálható Java
klienskönyvtárnál az Apache-2.0 tisztább vállalati szerződést ad.

Why not MIT? MIT is shorter and permissive, but it does not contain the same explicit,
detailed patent terms. Apache-2.0 gives adopters of a reusable Java library a clearer
enterprise-facing license contract.

A projekt nem kettős licencű. A felhasználónak nem kell MIT és Apache között
választania: egyetlen licenc, az Apache-2.0 érvényes.

This is not a dual-licensed project. Apache-2.0 is the sole project license.
