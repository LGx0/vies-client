- For brugsspørgsmål eller ideer, brug GitHub Discussion, hvis det er aktiveret.
- Rapporter kun en sikkerhedsfejl i henhold til [SECURITY.md](SECURITY.md).
- Brug fejlskabelonen til reproducerbare defekter.
- Brug GitHub Discussions til brugsspørgsmål, når det er aktiveret.
- Rapportér kun sårbarheder gennem [SECURITY.md](SECURITY.md).
