# English (en) — Security policy
> [Language selector](../../LANGUAGES.md) · English source governs; root `LICENSE` is authoritative.

Latest `1.x` receives fixes; older versions case by case. Report privately through **Security → Advisories → Report a vulnerability**, never a public issue or pre-fix exploit. Include version, environment, reproduction, impact, minimal non-personal proof and workaround. Targets: acknowledge within 7 days, assess within 14; these are not an SLA. Scope includes injection, endpoint/TLS bypass, data exposure, cache poisoning, requester misuse, unbounded resources, shutdown deadlock and false `Invalid`. Upstream VIES availability/throttling/data quality alone is out of scope.
