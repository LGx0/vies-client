# Eesti (et) — Turvapoliitika
> [Keelevalik](../../LANGUAGES.md) · Määravad ingliskeelne tehniline ja õiguslik originaal ning juure `LICENSE`/`NOTICE`.

Uusim `1.x` saab turvaparandusi, vanemad versioonid juhtumipõhiselt. Ära ava avalikku issue't ega avalda exploiti enne parandust. Teata GitHubi **Security → Advisories → Report a vulnerability** kaudu: versioon, keskkond, reproduktsioon, mõju, isikuandmeteta PoC ja workaround. Siht on kinnitus 7 ja esmane hinnang 14 päevaga; see pole SLA. Scope: injection, TLS/endpoint bypass, andmeleke, cache poisoning, requester väärkasutus, piiramatu ressursikasv, shutdown deadlock ja tehnilisest veast vale `Invalid`. VIES-i upstream katkestus/throttling/andmekvaliteet üksi pole kliendi haavatavus.
