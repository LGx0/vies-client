# Suomi (fi) — Tietoturvakäytäntö
> [Kielivalitsin](../../LANGUAGES.md) · Englanninkielinen tekninen ja oikeudellinen alkuperä sekä juuren `LICENSE`/`NOTICE` ovat määrääviä.

Uusin `1.x` saa tietoturvakorjaukset; vanhemmat versiot tapauskohtaisesti. Älä avaa julkista issuea tai julkaise exploitia ennen korjausta. Raportoi GitHubissa **Security → Advisories → Report a vulnerability**. Ilmoita versio, ympäristö, toisto, vaikutus, henkilötiedoton PoC ja workaround. Tavoitteet ovat vastaanottokuittaus 7 päivässä ja ensiarvio 14 päivässä; ne eivät ole SLA. Scopeen kuuluvat injection, TLS/endpoint-ohitus, tietovuoto, cache poisoning, requester-väärinkäyttö, rajaton resurssikasvu, shutdown-deadlock ja teknisestä viasta syntyvä väärä `Invalid`. Upstream VIESin katko, throttling tai jäsenmaan datan laatu yksin ei ole kirjaston haavoittuvuus.
