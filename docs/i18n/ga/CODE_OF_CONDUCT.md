# Gaeilge (ga) — Code of Conduct
> [Roghnóir teanga](../../LANGUAGES.md) · Is iad an bunleagan Béarla agus `LICENSE`/`NOTICE` na fréimhe atá i réim.

Pobal oscailte, gairmiúil, saor ó chiapadh beag beann ar thaithí, cúlra, féiniúlacht, cumas nó dearcadh. Bí measúil, sonrach agus cuiditheach; cáin smaoineamh, glac correction, cuidigh le newcomers, cosain eolas íogair. Tá harassment, threat, discrimination, doxxing, unwanted sexual attention, spam, deception, irresponsible disclosure agus false authorship toirmiscthe. Tuairiscigh go príobháideach; féadfaidh maintainers warning, content removal nó restriction a chur i bhfeidhm go rúnda/comhsheasmhach. Baineann sé le issue, PR, discussion, review, release agus ionadaíocht.
