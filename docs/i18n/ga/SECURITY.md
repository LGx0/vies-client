# Gaeilge (ga) — Security
> [Roghnóir teanga](../../LANGUAGES.md) · Is iad an bunleagan Béarla agus `LICENSE`/`NOTICE` na fréimhe atá i réim.

Faigheann `1.x` is déanaí fixes; leaganacha níos sine cás ar chás. Ná hoscail issue poiblí ná exploit roimh fix. Tuairiscigh go príobháideach trí **Security → Advisories → Report a vulnerability** le version, environment, reproduction, impact, PoC gan sonraí pearsanta agus workaround. Spriocanna 7 lá acknowledgement, 14 lá assessment; ní SLA iad. Scope: injection, TLS/endpoint bypass, data leak, cache poisoning, requester misuse, resource growth gan teorainn, shutdown deadlock agus `Invalid` bréagach. Ní vulnerability cliaint é outage/throttling/data quality VIES amháin.
