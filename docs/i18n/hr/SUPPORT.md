## Tehnička podrška

- Za ponovljive pogreške koristite predložak izvješća o pogrešci.
- Za pitanja ili ideje o korištenju koristite GitHub diskusiju, ako je omogućena.
- Prijavite sigurnosnu pogrešku samo prema [SECURITY.md](SECURITY.md).
- Koristite predložak grešaka za ponovljive nedostatke.
- Koristite GitHub rasprave za pitanja o korištenju kada je omogućeno.
- Prijavite ranjivosti samo putem [SECURITY.md](SECURITY.md).

Podrška zajednice je najbolji pokušaj, nezajamčeno vrijeme odgovora i nema zamjene
porezni, pravni ili računovodstveni savjet. Za poslovnu interpretaciju VIES odgovora i
integrirajuća strana odgovorna je za usklađenost s lokalnim zakonodavstvom.
