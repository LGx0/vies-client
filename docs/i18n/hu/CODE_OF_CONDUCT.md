# Magyar (hu) — Magatartási kódex
> [Nyelvválasztó](../../LANGUAGES.md) · Az angol eredeti irányadó.

Nyitott, szakmai, zaklatásmentes közösséget tartunk fenn minden tapasztalat, háttér, identitás, képesség és nézet számára. Légy tiszteletteljes, konkrét, konstruktív; ötletet kritizálj, fogadj javítást, segíts újoncnak és védd az érzékeny adatot. Zaklatás, fenyegetés, diszkrimináció, doxxolás, kéretlen szexuális figyelem, spam, megtévesztés, felelőtlen disclosure és hamis szerzőség tilos. Privát jelentés után a maintainerek figyelmeztethetnek, törölhetnek vagy korlátozhatnak, bizalmasan és következetesen. Minden issue/PR/discussion/review/release/projektképviseletre érvényes.
