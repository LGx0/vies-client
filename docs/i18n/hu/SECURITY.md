# Magyar (hu) — Biztonsági szabályzat
> [Nyelvválasztó](../../LANGUAGES.md) · Az angol eredeti és a gyökér `LICENSE` irányadó.

A legfrissebb `1.x` kap javítást, régebbi verziók eseti alapon. Jelentés kizárólag privátan: **Security → Advisories → Report a vulnerability**; ne nyiss publikus issue-t és ne publikálj exploitot. Add meg a verziót, környezetet, reprodukciót, hatást, személyes adat nélküli PoC-t és workaroundot. Cél: 7 napos visszaigazolás, 14 napos első értékelés, nem SLA. Releváns: injection, TLS/endpoint bypass, adatszivárgás, cache poisoning, requester misuse, korlátlan erőforrás, shutdown deadlock, téves `Invalid`. Az upstream kiesése/throttlingja/adatminősége önmagában nem sérülékenység.
