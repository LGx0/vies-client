# Lietuvių (lt) — Code of Conduct
> [Kalbų pasirinkimas](../../LANGUAGES.md) · Pirmenybę turi angliškas techninis/teisinis originalas ir šakniniai `LICENSE`/`NOTICE`.

Kuriame atvirą, profesionalią ir be priekabiavimo bendruomenę nepriklausomai nuo patirties, kilmės, tapatybės, gebėjimų ar požiūrio. Būkite pagarbūs, konkretūs, konstruktyvūs; kritikuokite idėją, priimkite taisymą, padėkite naujokams, saugokite jautrią informaciją. Priekabiavimas, grasinimai, diskriminacija, doxxing, nepageidaujamas seksualinis dėmesys, spam, apgaulė, neatsakingas disclosure ir melaginga autorystė draudžiami. Praneškite privačiai; maintainers gali įspėti, šalinti turinį ar riboti dalyvavimą, konfidencialiai ir nuosekliai. Taikoma issue, PR, discussions, reviews, releases ir projekto atstovavimui.
