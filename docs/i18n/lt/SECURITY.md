# Lietuvių (lt) — Security
> [Kalbų pasirinkimas](../../LANGUAGES.md) · Pirmenybę turi angliškas techninis/teisinis originalas ir šakniniai `LICENSE`/`NOTICE`.

Naujausia `1.x` gauna saugumo pataisas; senesnės versijos – atskiru sprendimu. Neatidarykite viešo issue ir neskelbkite exploit iki pataisos. Praneškite privačiai per **Security → Advisories → Report a vulnerability**: versija, aplinka, atkūrimas, poveikis, PoC be asmens duomenų, workaround. Tikslai: patvirtinti per 7, įvertinti per 14 dienų; tai ne SLA. Scope: injection, TLS/endpoint bypass, duomenų nutekėjimas, cache poisoning, requester misuse, neriboti resursai, shutdown deadlock, klaidingas `Invalid`. Upstream VIES sutrikimas/throttling/duomenų kokybė savaime nėra pažeidžiamumas.
