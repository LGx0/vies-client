# Latviešu (lv) — Code of Conduct
> [Valodu izvēle](../../LANGUAGES.md) · Noteicošais ir angļu tehniskais/juridiskais oriģināls un saknes `LICENSE`/`NOTICE`.

Veidojam atvērtu, profesionālu, no uzmākšanās brīvu kopienu neatkarīgi no pieredzes, izcelsmes, identitātes, spējām vai uzskata. Esi cieņpilns, konkrēts un konstruktīvs; kritizē ideju, pieņem labojumu, palīdzi iesācējiem, sargā sensitīvu informāciju. Uzmākšanās, draudi, diskriminācija, doxxing, nevēlama seksuāla uzmanība, spam, maldināšana, bezatbildīga disclosure un nepatiesa autorība nav pieļaujama. Ziņo privāti; maintainers var brīdināt, dzēst saturu vai ierobežot dalību, konfidenciāli un konsekventi. Attiecas uz issue, PR, discussions, reviews, releases un projekta pārstāvību.
