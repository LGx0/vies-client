# Latviešu (lv) — Security
> [Valodu izvēle](../../LANGUAGES.md) · Noteicošais ir angļu tehniskais/juridiskais oriģināls un saknes `LICENSE`/`NOTICE`.

Jaunākais `1.x` saņem labojumus; vecāki laidieni pēc atsevišķa lēmuma. Neatver publisku issue un nepublicē exploit pirms labojuma. Ziņo privāti: **Security → Advisories → Report a vulnerability**; norādi versiju, vidi, reprodukciju, ietekmi, PoC bez personas datiem un workaround. Mērķi: apstiprinājums 7, sākotnējs vērtējums 14 dienās; tas nav SLA. Scope: injection, TLS/endpoint bypass, datu noplūde, cache poisoning, requester ļaunprātīga izmantošana, neierobežoti resursi, shutdown deadlock un kļūdains `Invalid`. Upstream VIES nepieejamība/throttling/datu kvalitāte pati par sevi nav ievainojamība.
