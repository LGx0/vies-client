## Suporte técnico / Suporte técnico

- Para erros reproduzíveis, use o modelo de relatório de bug.
- Para dúvidas ou ideias de uso, use GitHub Discussion, se habilitado.
- Relate um erro de segurança apenas de acordo com [SECURITY.md](SECURITY.md).
- Use o modelo de bug para defeitos reproduzíveis.
- Use as discussões do GitHub para perguntas de uso quando habilitado.
- Reportar vulnerabilidades apenas através de [SECURITY.md](SECURITY.md).

aconselhamento fiscal, jurídico ou contabilístico. Para a interpretação comercial da resposta VIES e da
o integrador é responsável pelo cumprimento da legislação local.

aconselhamento jurídico ou contábil. Os integradores continuam responsáveis ​​pelas decisões de negócios
