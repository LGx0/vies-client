## Suport tehnic

- Pentru erori reproductibile, utilizați șablonul de raport de erori.
- Pentru întrebări sau idei de utilizare, utilizați GitHub Discussion, dacă este activat.
- Raportați o eroare de securitate numai conform [SECURITY.md](SECURITY.md).
- Utilizați șablonul de erori pentru defecte reproductibile.
- Folosiți Discuții GitHub pentru întrebări de utilizare atunci când este activat.
- Raportați vulnerabilități numai prin [SECURITY.md](SECURITY.md).
