- För reproducerbara fel, använd felrapportmallen.
- För användningsfrågor eller idéer, använd GitHub Discussion, om aktiverat.
- Rapportera ett säkerhetsfel endast enligt [SECURITY.md](SECURITY.md).
- Använd buggmallen för reproducerbara defekter.
- Använd GitHub Discussions för användningsfrågor när det är aktiverat.
- Rapportera sårbarheter endast via [SECURITY.md](SECURITY.md).
